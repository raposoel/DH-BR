# DH-BR
Um projeto de tradução paralelo para o Foundry, deixando o jogo mais próximo do original. Sem inventar, só traduzir.
<br>
<br>Diferenças com a tradução oficial:
- Retirada de "Pontos de", mantendo o texto mais próximo do tom literário original.
- Preservar o tom evocativo e leve do original, que frequentemente utiliza construções referênciando a ficha física, como “marque uma Esperança” ou “limpe um Estresse”.
- Ajuste de termos traduzidos de maneira duvidosa, como "Fadiga".
- Preservar o estilo do texto original para manter a integridade da obra.

Atualizado para a versão 2.4.0

## Problemas Identificados:

### A Forma de Fera do Druida.
Para que a tradução funcione, é preciso manual colocar as pastas do compêndio Formas de Fera na aba de Itens. 

![Passo 01](https://github.com/raposoel/DH-BR/blob/main/imagens/ex01.webp?raw=true)
![Passo 02](https://github.com/raposoel/DH-BR/blob/main/imagens/ex02.webp?raw=true)
![Passo 03](https://github.com/raposoel/DH-BR/blob/main/imagens/ex03.webp?raw=true)

Esses mesmos Itens de forma de fera ali podem ser usados para a troca de imagens para o druida.
Na versão final, haverá uma Macro para fazer isso automaticamente.

### Invocações.
As magias que invocam seres/coisas foram melhoradas, atribuindo um Ator genérico para representar a invocação. Para personalizar a sua invocação, na ficha do personagem, é preciso criar um novo Ator do Zero, configurar da maneira que achar melhor (ou só copie o que foi feito no meu Ator) e editar o poder trocando a ficha a ser invocada.

O sistema acaba por limitar duplicatas ou cópias, então só funciona dessa maneira.

Copie a pasta de invocações do Compendium para a aba de atores, e coloque os jogadores como donos da pasta.

### Para melhores resultados, delete os compêndios de "Character Options" em inglês no seu mundo, caso possível.
